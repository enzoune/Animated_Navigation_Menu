# 🧭 Animated Navigation Bar (CSS-Only Slider)

> Uma barra de navegação moderna com efeito de "slider" animado, desenvolvida exclusivamente com HTML5 e CSS3, focada em performance e experiência do usuário (UX).

## 📖 Sobre o Projeto

Este projeto demonstra a criação de uma **Nav Bar Animada** funcional, onde o indicador de seleção (slider) desliza suavemente entre os itens do menu. O diferencial técnico aqui é o uso da lógica de **Inputs de Rádio** para controlar a posição do slider e o estado ativo de cada link, eliminando a necessidade de JavaScript para a alternância visual.

## ✨ Destaques e Aprendizados Técnicos

* **Gerenciamento de Estado Nativo:** Utilização de `input type="radio"` para rastrear qual item da navegação está selecionado.
* **Interatividade com Label:** Uso estratégico de tags `<label>` vinculadas aos inputs para permitir que o clique no texto ou ícone dispare a transição.
* **Animação de Slider:** Estrutura preparada para animações de transição suave (`transition`) conforme o usuário navega entre as seções: Início, Blog, Código, Ajuda e Sobre.
* **Arquitetura de Ícones:** Integração da biblioteca Font Awesome (v5.15.2) para uma interface visualmente rica e semântica.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação de formulário invisível para lógica de navegação.
* **CSS3:** Estilização de layout (Flexbox/Grid), transições dinâmicas e design responsivo.
* **Font Awesome:** Biblioteca de ícones para auxílio visual na navegação.
